Stage your changes
Tell Git which files you want to include in the next snapshot.

Bash
git add .
(The . adds all changed files. You can also specify a file name like git add index.html)

Commit the changes
Save the snapshot locally with a descriptive message.

Bash
git commit -m "Briefly describe what you changed"
Push to GitHub
Send those local commits to the online repository.

Bash
git push origin main
